## Compilazione

I file sono scritti in markdown, per ottenere i .pdf basta utilizzare Pandoc.

Per i file con le spiegazioni  
`pandoc input.md -o output.pdf  

Per le presentazioni  
`pandoc -t beamer input.md -o output.pdf
